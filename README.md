## About The Project

<p align="center">
  <img src="https://github.com/hues0s/watchWeather/blob/main/Screenshots/demo.png" width="100%" />
</p>

Welcome to my [watchWeather](https://github.com/hues0s/watchWeather) MVVM demo!

### Features
* Reactive programming: Built with [SwiftUI](https://developer.apple.com/documentation/swiftui/) and [Combine](https://developer.apple.com/documentation/combine) following the MVVM pattern
* CoreLocation: Get user's current location
* OpenWeatherMap API: Built using their [Current Weather Data API](https://openweathermap.org/current)
* Async / Await

### Requirements
- watchOS 6.0 or later
